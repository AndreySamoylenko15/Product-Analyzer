# 🛒 Product Analyzer (Python)

Простий Python-скрипт для аналізу списку продуктів та їх цін.

## 🚀 Можливості

Програма виконує:

* 💰 Підрахунок загальної вартості всіх продуктів
* 📈 Пошук найдорожчого продукту
* 📉 Пошук найдешевшого продукту

---

## 🧾 Вхідні дані

Продукти задаються у вигляді словника:

```python
products = {
    'apple': 10,
    'banana': 15,
    'milk': 30,
    'bread': 25
}
```

---

## 🛠 Як запустити

```bash
python product_analyzer.py
```

---

## 📌 Приклад результату

```
Total price: 80
Most expensive: ('milk', 30)
Cheapest product: ('apple', 10)
```

---

## ⚠️ Важливий момент

Функція пошуку найдешевшого продукту повинна коректно ініціалізувати мінімальне значення:

❌ Неправильно:

```python
min_price = 0
```

✔ Правильно:

```python
min_price = float('inf')
```

* базових алгоритмів пошуку
